第一章 绪论

第一章 绪论

自动控制系统的基本组成与反馈思想

先把系统看成“比较目标和输出，再自动修正偏差”的闭环。能分清输入、输出、扰动、反馈、控制器和被控对象，是后面列传递函数与画框图的前置条件。

前置知识

  • 函数与信号的输入输出关系
  • 反馈的正负号
  • 稳态与动态的直观含义

必须掌握

  • 闭环控制的核心是用反馈量与给定量比较得到偏差，再由控制器产生控制作用。
  • 评价控制系统通常看稳定性、快速性、准确性。
  • 开环结构简单但抗扰能力弱；闭环能修正偏差，但必须额外关注稳定性。

易错点

  • 只背定义不看信号流向，导致后面方框图等效变换时符号出错。

第二章 数学模型

第二章 数学模型

Laplace 变换与反变换

第二章作业先考基本变换。要先熟悉常见函数的 Laplace 变换，再用线性性质、位移性质和部分分式展开做反变换。

前置知识

  • 指数、三角函数基本形式
  • 复数与极点
  • 部分分式展开

必须掌握

  • 微分方程可以通过 Laplace 变换转为代数方程。
  • 反变换常用部分分式展开；遇到二阶因子时先配方，再对应衰减正弦/余弦。
  • 初始条件会进入输出响应计算，不能默认全为零。

公式/算式

L{e^{-a t}} = 1 / (s + a)
L{e^{-a t} cos wt} = (s + a) / ((s + a)^2 + w^2)
L{e^{-a t} sin wt} = w / ((s + a)^2 + w^2)

易错点

  • 把 s 平移写反，例如 e^{-a t} 对应 s+a，不是 s-a。

第二章 数学模型

传递函数、结构图与信号流图

传递函数描述零初始条件下输出与输入的比值。框图化简和梅逊公式是把复杂连接关系转成闭环传递函数的工具。

前置知识

  • Laplace 变换
  • 代数方程消元
  • 负反馈公式

必须掌握

  • 典型负反馈闭环传递函数：前向通道除以一加开环环路传递函数。
  • 系统型别由开环传递函数中原点极点个数决定。
  • 信号流图用前向通路、回路和不接触回路计算总传递函数。

公式/算式

Phi(s) = G(s) / (1 + G(s)H(s))
Mason: T = sum(P_k Delta_k) / Delta

易错点

  • 把闭环极点当成开环极点；判断稳定性应看闭环特征方程。

第三章 时域分析

第三章 时域分析

一阶、二阶系统的时间响应指标

第三章作业的核心是从响应曲线或传递函数读出自然频率、阻尼比、超调量、峰值时间和调节时间。

前置知识

  • 传递函数
  • 极点与响应形式
  • 指数衰减与正弦振荡

必须掌握

  • 标准二阶系统由自然频率和阻尼比决定动态性能。
  • 欠阻尼时会出现超调和振荡；阻尼比越大，超调越小。
  • 调节时间常按 5% 或 2% 误差带近似，做题时要看教材或题目约定。

公式/算式

G(s) = wn^2 / (s^2 + 2 zeta wn s + wn^2)
wd = wn sqrt(1 - zeta^2)
sigma% = exp(-zeta pi / sqrt(1 - zeta^2)) x 100%
tp = pi / wd
ts ≈ 3 / (zeta wn) (5% criterion)

易错点

  • 把阻尼振荡频率 wd 和自然频率 wn 混用。

第三章 时域分析

稳定性、劳斯判据与稳态误差

稳定性看闭环极点是否都在左半平面；稳态误差看系统型别和静态误差系数。先会写闭环特征方程，再套劳斯表和终值定理。

前置知识

  • 闭环特征方程
  • 极点位置
  • 终值定理

必须掌握

  • 劳斯判据通过首列符号判断右半平面根个数。
  • 稳态误差通常用终值定理计算，也可用位置、速度、加速度误差系数快速判断。
  • 输入类型不同，稳态误差结论不同：阶跃、斜坡、抛物线不能混用。

公式/算式

ess = lim_{s -> 0} s E(s)
Kp = lim_{s -> 0} G(s)H(s)
Kv = lim_{s -> 0} s G(s)H(s)
Ka = lim_{s -> 0} s^2 G(s)H(s)

易错点

  • 用开环稳定替代闭环稳定；静态误差系数没有按系统型别判断。

第五章 频域分析

第五章 频域分析

频率特性、幅相曲线与 Bode 图

频域分析把 s 换成 jw，研究不同频率正弦输入下系统输出幅值和相位如何变化。Bode 图本质是把幅频和相频用对数坐标画出来。

前置知识

  • 复数模与幅角
  • 传递函数
  • 对数坐标和 dB

必须掌握

  • 幅频特性是 |G(jw)|，相频特性是 angle G(jw)。
  • Bode 幅值图由低频基准线和各典型环节转折频率叠加得到。
  • 积分环节给 -20 dB/dec 和 -90° 相位；一阶惯性环节过转折频率后斜率再降 -20 dB/dec。

公式/算式

G(jw) = G(s)|_{s = jw}
L(w) = 20 log10 |G(jw)|

易错点

  • 画 Bode 图时没有先化成尾 1 标准型，导致转折频率和增益错位。

第五章 频域分析

Nyquist 判据、对数稳定判据与稳定裕度

频域稳定性不是只看曲线形状，而是看开环频率特性怎样绕过临界点 -1。稳定裕度则衡量离临界失稳还有多远。

前置知识

  • 复平面
  • 开环传递函数
  • 闭环稳定的特征方程

必须掌握

  • Nyquist 判据通过开环频率特性包围 -1 点的情况判断闭环稳定。
  • 相角裕度和幅值裕度分别从穿越频率读出。
  • 裕度越大通常稳定余量越充分，但过大可能牺牲快速性。

公式/算式

gamma = 180° + phase G(j wc)
Kg = 1 / |G(j wg)|

易错点

  • 把幅值穿越频率和相位穿越频率互换。

第五章 频域分析

由开环/闭环频域指标分析系统性能

低频段影响稳态精度，中频段影响动态性能，高频段影响抗噪。复习时要能把曲线形态和性能语言互相转换。

前置知识

  • Bode 图
  • 稳定裕度
  • 时域性能指标

必须掌握

  • 低频增益高通常稳态误差小。
  • 中频段斜率和穿越频率影响快速性与稳定裕度。
  • 闭环频域指标包括谐振峰值、谐振频率和带宽。

公式/算式

Mr = max |Phi(jw)|
wb: |Phi(jw)| drops to 0.707 of low-frequency value

易错点

  • 只提高增益追求精度，却忽略稳定裕度下降。

第六章 校正

第六章 校正

超前校正与滞后校正

校正是在原系统性能不满足要求时增加环节。超前校正常用于增加相角裕度和改善动态性能；滞后校正常用于改善稳态精度并尽量少破坏中频段。

前置知识

  • Bode 图
  • 稳定裕度
  • 稳态误差系数

必须掌握

  • 超前校正提供正相角，通常把最大超前角放在新的穿越频率附近。
  • 滞后校正提高低频增益，转折频率要放在较低频段。
  • 设计题必须按指标反推参数，而不是随意添加环节。

公式/算式

Gc_lead(s) = (T s + 1) / (alpha T s + 1), 0 < alpha < 1
Gc_lag(s) = (beta T s + 1) / (T s + 1), beta > 1

易错点

  • 超前/滞后参数条件记反。

第八章 非线性

第八章 非线性

非线性系统与描述函数法

非线性系统不能简单套叠加原理。描述函数法用近似频域方法分析某些非线性环节引起的自振或极限环。

前置知识

  • 频率响应
  • 相平面基本概念
  • 线性化近似

必须掌握

  • 常见非线性包括饱和、死区、继电、滞环等。
  • 描述函数法是近似方法，适合特定非线性和近似正弦运动。
  • 非线性系统可能有多个平衡点和极限环。

易错点

  • 把非线性系统当作普通线性系统直接叠加。

第九章 状态空间

第九章 状态空间

状态空间描述、可控性与可观测性

状态空间用一组一阶微分方程描述系统内部状态，适合多输入多输出系统。先会选状态变量，再写 A、B、C、D 矩阵。

前置知识

  • 矩阵乘法
  • 线性代数秩
  • 微分方程

必须掌握

  • 标准形式：x_dot = A x + B u，y = C x + D u。
  • 可控性看可控矩阵秩；可观测性看可观测矩阵秩。
  • 状态反馈和状态观测器建立在可控、可观测条件上。

公式/算式

x_dot = A x + B u
y = C x + D u
Qc = [B, AB, ..., A^{n-1}B]
Qo = [C; CA; ...; CA^{n-1}]

易错点

  • 只写传递函数，不会转换成状态变量形式；矩阵维度不匹配。
